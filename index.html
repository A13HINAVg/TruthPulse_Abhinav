About TruthPulse

        
TruthPulse is an open-source fake news detection tool built with FastAPI and vanilla JS. It uses heuristic NLP signals to classify content and supports multilingual input via Google Translate.

        
⚠️ No automated system is perfect. Always verify important claims through primary sources. TruthPulse is an evidence-assisted aid, not a definitive arbiter of truth.
